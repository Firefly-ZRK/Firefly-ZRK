#### 🔧 轻量化智能图像去噪算法研发 | `独立研发`
*   **赛事:** 中兴捧月大赛
*   **时间:** 2025年3月 - 2025年5月
*   **工作内容:**
    *   项目针对复杂场景（逆光、低光照、运动模糊等）下的图像去噪与清晰度提升任务，在30G MACs算力约束（720P输入）下设计算法，满足PSNR与视觉质量双重要求。

*   **成就:** 在官方测试集实现PSNR 22.85dB、SSIM 0.902，MACs 29.3G，**综合排名第7**。
